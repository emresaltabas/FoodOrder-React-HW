## Setup

- In the project directory, ou can follow the steps below to download the dependencies:
  - Install with npm:
  ```sh
  npm install
  ```
  - Install with yarn:
  ```sh
  yarn install
  ```
- Run React dev server (http://localhost:5173/)
  - Run with npm:
  ```sh
  npm run dev
  ```
  - Run with yarn:
  ```sh
  yarn run dev
   ```
